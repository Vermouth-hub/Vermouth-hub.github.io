example1: 跳跃游戏

本题使用贪心算法。通过分析可知，对于数组中的任意位置y，如果存在x，使得x + nums[x] >= y，则位置y是可达的。
因此，我们只需要依次维护最远可到达的位置max_distance，如果当前位置i <= max_distance，则说明位置i是可达的，我们可以更新max_distance = max(max_distance, i + nums[i])。
最终，如果max_distance >= n - 1，则说明最后一个位置是可达的，否则不可达。
